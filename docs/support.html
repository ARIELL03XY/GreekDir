Preguntas frecuentes

Algunas carpetas aparecen con 🔒 "Sin acceso"

En macOS, el sistema protege carpetas como Documentos, Escritorio o Mail. Para un escaneo completo, activa Acceso total al disco para GreekDir en Ajustes del Sistema → Privacidad y seguridad.

El total escaneado no coincide con el uso del disco

Por defecto GreekDir omite carpetas ocultas y de sistema. Marca "Incluir carpetas ocultas y de sistema" en la pantalla de inicio para contarlas.

Frequently asked questions

Some folders show 🔒 "No access"

On macOS, the system protects folders like Documents, Desktop, or Mail. For a complete scan, enable Full Disk Access for GreekDir in System Settings → Privacy & Security.

The scanned total doesn't match disk usage

By default GreekDir skips hidden and system folders. Check "Include hidden and system folders" on the start screen to count them.
